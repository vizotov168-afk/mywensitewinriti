Формулы сокращенного умножения (ФСУ)

Формулы сокращенного умножения позволяют упрощать алгебраические выражения, выполнять умножение многочленов быстрее и решать уравнения.

Основные формулы:

1. Квадрат суммы: (a + b)² = a² + 2ab + b²
2. Квадрат разности: (a - b)² = a² - 2ab + b²
3. Разность квадратов: a² - b² = (a - b)(a + b)
4. Куб суммы: (a + b)³ = a³ + 3a²b + 3ab² + b³
5. Куб разности: (a - b)³ = a³ - 3a²b + 3ab² - b³

Пример 1: Раскрытие скобок

Раскройте скобки: (2x + 3y)²

Решение: Используем формулу квадрата суммы:

(2x + 3y)² = (2x)² + 2·2x·3y + (3y)² = 4x² + 12xy + 9y²

Пример 2: Разложение на множители

Разложите на множители: 25x² - 49y²

Решение: Это разность квадратов: 25x² = (5x)², 49y² = (7y)²

25x² - 49y² = (5x - 7y)(5x + 7y)

Пример 3: Разные формы ответа

Для выражения 4x² - 144 можно записать:

1. Разложенная форма: (2x - 12)(2x + 12)

2. Стандартная форма: 4x² - 144

3. Альтернативная форма: (2x)² - 12²

💡 Все эти формы считаются правильными!

Задачи на сплавы, смеси и растворы

Эти задачи относятся к текстовым задачам на проценты и часто встречаются в экзаменационных работах.

Основные понятия:

Концентрация — процентное содержание вещества в смеси.

Масса вещества = масса смеси × концентрация / 100

Пример 1: Смешивание двух растворов

Дано:

  • Первый раствор: масса = 40 кг, концентрация = 12%
  • Второй раствор: концентрация = 20%
  • Нужно получить: концентрация = 15%

Найти: массу второго раствора

Решение: Пусть нужно добавить x кг 20% раствора.

Составляем уравнение: 40·0.12 + x·0.20 = (40 + x)·0.15

4.8 + 0.20x = 6 + 0.15x

0.05x = 1.2

x = 24 кг

Пример 2: Выпаривание воды

Дано:

  • Исходный раствор: масса = 50 кг, концентрация = 8%
  • После выпаривания: концентрация = 20%

Найти: массу выпаренной воды

Решение: Масса соли постоянна: 50·0.08 = 4 кг

После выпаривания: 4 кг соли составляют 20% раствора

Масса нового раствора = 4 / 0.20 = 20 кг

Выпарили воды: 50 - 20 = 30 кг

💡 Совет: В задачах на сплавы всегда определяйте, что остается постоянным (обычно масса чистого вещества или масса металла).

Линейные уравнения и их решение

Линейное уравнение — уравнение вида ax + b = 0, где a и b — числа, x — переменная.

Алгоритм решения:

  1. Раскрыть скобки
  2. Перенести слагаемые с переменной в одну часть, числа — в другую
  3. Привести подобные слагаемые
  4. Разделить обе части на коэффициент при переменной

Пример 1: Простое уравнение

Решите уравнение: 5x - 12 = 3x + 4

Решение:

5x - 3x = 4 + 12

2x = 16

x = 8

Пример 2: Уравнение со скобками

Решите уравнение: 3(x - 4) = 5(2x + 1) - 7

Решение:

Раскрываем скобки: 3x - 12 = 10x + 5 - 7

3x - 12 = 10x - 2

3x - 10x = -2 + 12

-7x = 10

x = -10/7 ≈ -1.43

Многочлены и действия с ними

Многочлен (полином) — сумма одночленов.

Стандартный вид многочлена:

Пример: 3x³ - 2x² + 5x - 7

Действия с многочленами:

1. Сложение многочленов:

(3x² + 2x - 5) + (x² - 4x + 3) = (3+1)x² + (2-4)x + (-5+3) = 4x² - 2x - 2

2. Вычитание многочленов:

(5x² - 3x + 2) - (2x² + x - 4) = 5x² - 3x + 2 - 2x² - x + 4 = 3x² - 4x + 6

3. Умножение многочлена на одночлен:

3x(2x² - 4x + 5) = 3x·2x² - 3x·4x + 3x·5 = 6x³ - 12x² + 15x

💡 Запомните: При сложении и вычитании многочленов складываются/вычитаются только коэффициенты при одинаковых степенях.

Пример: Разность квадратов

  • (2x - 12)(2x + 12) ✓

  • 4x² - 144 ✓

Задача: Раскройте скобки: (x + 5)²

Можно ввести:

  • x^2 + 10x + 25 ✓
  • x² + 10x + 25 ✓
  • 1x^2 + 10x + 25 ✓

Пример: Уравнение

Задача: Решите уравнение: 3(x - 2) = 1(x + 3)

Можно ввести:

  • 4.5 ✓ (десятичная дробь)
  • 9/2 ✓ (обыкновенная дробь)
  • 4,5 ✓ (с запятой)
