Формулы сокращенного умножения

Формулы сокращенного умножения (ФСУ) — это математические тождества, которые позволяют быстро выполнять умножение многочленов и раскладывать их на множители.

(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
a² - b² = (a - b)(a + b)
(a + b)³ = a³ + 3a²b + 3ab² + b³
(a - b)³ = a³ - 3a²b + 3ab² - b³

📌 Примеры:

(2x + 3)² = (2x)² + 2·2x·3 + 3² = 4x² + 12x + 9

(5x - 2)² = (5x)² - 2·5x·2 + 2² = 25x² - 20x + 4

9x² - 16 = (3x)² - 4² = (3x - 4)(3x + 4)

💡 Запомните: Формулы сокращенного умножения помогают быстро раскрывать скобки и раскладывать на множители. Это основа для решения многих уравнений и упрощения выражений!

Задачи на сплавы и смеси

Задачи на сплавы и смеси — это задачи на проценты, где нужно найти массу вещества, концентрацию или массу раствора.

Основная формула:
масса вещества = масса смеси × концентрация / 100

📌 Пример 1: Смешивание растворов

Условие: Смешали 40 кг 12% раствора соли и 20% раствор. Получили 15% раствор. Сколько добавили 20% раствора?

Решение:

1. Масса соли в 1-м растворе: 40 × 0.12 = 4.8 кг

2. Пусть x кг — масса 2-го раствора. Масса соли в нем: 0.2x кг

3. Уравнение: 4.8 + 0.2x = 0.15(40 + x)

4. 4.8 + 0.2x = 6 + 0.15x → 0.05x = 1.2 → x = 24 кг

📌 Пример 2: Выпаривание

Условие: Из 50 кг 8% раствора выпарили воду и получили 20% раствор. Сколько воды выпарили?

Решение:

1. Масса соли: 50 × 0.08 = 4 кг (постоянна)

2. Масса нового раствора: 4 / 0.20 = 20 кг

3. Выпарили воды: 50 - 20 = 30 кг

💡 Совет: В задачах на сплавы всегда определяйте, что остается постоянным (обычно масса чистого вещества). Составляйте уравнение, где слева и справа — масса вещества.

Линейные уравнения

Линейное уравнение — это уравнение вида ax + b = 0, где a ≠ 0. Решение линейного уравнения сводится к нахождению значения переменной x.

Алгоритм решения:
1. Раскрыть скобки
2. Перенести переменные в левую часть, числа — в правую
3. Привести подобные слагаемые
4. Разделить обе части на коэффициент при переменной

📌 Пример 1: Простое уравнение

5x - 12 = 3x + 4

5x - 3x = 4 + 12

2x = 16

x = 8

📌 Пример 2: Уравнение со скобками

3(x - 2) = x + 3

3x - 6 = x + 3

3x - x = 3 + 6

2x = 9

x = 4.5

💡 Запомните: При переносе слагаемых из одной части уравнения в другую знак меняется на противоположный!

Многочлены

Многочлен (полином) — это сумма одночленов. Пример: 3x² - 2x + 5

📌 Сложение многочленов

(3x² + 2x - 5) + (x² - 4x + 3)

= (3+1)x² + (2-4)x + (-5+3)

= 4x² - 2x - 2

📌 Вычитание многочленов

(5x² - 3x + 2) - (2x² + x - 4)

= 5x² - 3x + 2 - 2x² - x + 4

= 3x² - 4x + 6

📌 Умножение многочленов

(x + 2)(x - 3)

= x·x + x·(-3) + 2·x + 2·(-3)

= x² - 3x + 2x - 6

= x² - x - 6

💡 Запомните: При сложении и вычитании многочленов складываются/вычитаются только коэффициенты при одинаковых степенях. При умножении — каждый член первого многочлена умножается на каждый член второго.
